# yearn-security

## Security policy
Yearn's security policy is available at https://github.com/yearn/yearn-security/blob/master/SECURITY.md

## About
This git repository is a central place for all other security-related information about the Yearn project. It includes the following:

* Security contact PGP keys (`keys/`)
* Public disclosures (`disclosures/`)
* Security audits (`audits/`)
* Security guides (`guides/`)
* Templates (`templates/`)

## Guides

Resources for security researchers and the Yearn security team:

* [Audit Checklist](guides/AUDIT_CHECKLIST.md) - Comprehensive smart contract security audit checklist
* [Severity Classification](guides/SEVERITY_CLASSIFICATION.md) - Vulnerability severity levels and classification guidelines
* [Emergency Response Runbook](guides/EMERGENCY_RESPONSE_RUNBOOK.md) - Incident response procedures and war room protocols

## Templates

Standardized templates for security processes:

* [Disclosure Template](templates/DISCLOSURE_TEMPLATE.md) - Template for writing incident disclosures
* [Bug Bounty Submission](templates/BUG_BOUNTY_SUBMISSION.md) - Template for submitting bug bounty reports

## Verify
The files contained in this repository can be verified in two ways:

* By verifying the git commit tags (`git tag -v`)
* By verifying the detached PGP signatures provided for most files included here
